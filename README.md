# AWS S3 Static Website Hosting Project

## Overview
This project hosts a personal portfolio website using AWS S3 and CloudFront.

## Technologies Used
- HTML
- CSS
- AWS S3
- AWS CloudFront
- Git & GitHub

## Steps Performed

1. Created a responsive portfolio website.
2. Created an S3 bucket.
3. Enabled Static Website Hosting.
4. Uploaded index.html and styles.css.
5. Configured bucket permissions.
6. Created a CloudFront distribution.
7. Enabled HTTPS through CloudFront.
8. Verified website deployment.
